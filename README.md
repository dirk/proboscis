# Proboscis

**A new meta-framework for RPC.**

## Transports

### TCP

The TCP transport is based off of the [Netstring][netstring]/[Bencode][bencode] formats.

  [netstring]: http://en.wikipedia.org/wiki/Netstring
  
  [bencode]: http://en.wikipedia.org/wiki/Bencode
